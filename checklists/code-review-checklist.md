# Code Review Checklist

## Before Review
- [ ] PR has clear title and description
- [ ] CI checks are passing
- [ ] PR is reasonably sized

## Code Quality
- [ ] Code follows style guidelines
- [ ] Code is readable
- [ ] No debug statements

## Testing
- [ ] New code has tests
- [ ] All tests pass
- [ ] Coverage maintained

## Documentation
- [ ] Code comments present
- [ ] README updated
- [ ] CHANGELOG updated
